Description of : metainformation

meta-tags with name-attribute:

The meta-tags are always contained within the head-element. The metadata provides information
about the data within the HTML-document, but is not visually displayed on the website.

author

Defines the author of the webpage.

description

Provides a description of the content of the webpage. i.e "Basic HTML5 guide"

generator

keywords

As in any report or such it is common to provide a list of keywords to tell the user what he/she can expect.

The http-equiv attribute provides an HTTP header for the information/value of the content attribute.
